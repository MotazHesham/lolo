Replay recorded session (rrweb)

How to run a recorded session

  1. Start the server: npm start
  2. Open this page: http://localhost:3000/replay.html
  3. Choose a session from the dropdown (sessions are in data/sessions/) and click “Load & replay”.

Each quiz completion saves one file: data/sessions/session-<id>-<timestamp>.json. The replay uses the rrweb recordedSessions[0].events from that file.
